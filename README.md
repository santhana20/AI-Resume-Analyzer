AI-Resume-Analyzer

AI-powered resume analyzer that scans your resume and gives instant ATS score, keyword match, and improvement suggestions. Built for job seekers to beat resume filters.

✨ Features
- **ATS Compatibility Check**: Checks if resume passes recruiter tracking systems
- **Keyword Analysis**: Matches resume with job description and highlights missing keywords  
- **Instant Suggestions**: AI tips for formatting, grammar, and wording improvements
- **Score Report**: Get score out of 100 with detailed feedback

🛠️ Tech Stack
`Python` `Flask` `HTML5` `CSS3` `JavaScript` `OpenAI API` `NLP` `PDF Processing`

🚀 Installation & Run
```bash
git clone https://github.com/santhana20/AI-Resume-Analyzer.git
cd AI-Resume-Analyzer
pip install -r requirements.txt
python app.py
App will run on `http://localhost:5000`

📖 Usage
1. Run `python app.py`
2. Open browser and go to http://localhost:5000
3. Upload your resume PDF/DOCX
4. Paste target job description
5. Get instant ATS score + improvement report
